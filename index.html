LSTM Time-Series Architecture

The Long Short-Term Memory (LSTM) network captures temporal degradation patterns:

  • Input: 72-hour sliding window of multivariate sensor data
  • Hidden: 2 stacked LSTM layers (128, 64 units) with dropout (0.3)
  • Attention: Self-attention to weight critical time steps
  • Output: Dense layer with sigmoid for failure probability [0,1]
# Kazmi Methodology - LSTM
model = Sequential([
    LSTM(128, return_sequences=True,
         input_shape=(window, features)),
    Dropout(0.3),
    LSTM(64),
    Dropout(0.3),
    Dense(32, activation='relu'),
    Dense(1, activation='sigmoid')
])
